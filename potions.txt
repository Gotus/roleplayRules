Данный файл содержит описание механики создания зелий.

Персонаж вместо привала может заняться сбором трав и изготовлением зелий. Сбор трав и сбор зелий происходит автоматически.
Результат этой процедуры зависит от местности, бросков кубика, навыка алхимии.
Первый бросок делается на количество изготовленных зелий. Число зелий определяется по следующей формуле:
(Бросок кубика) / 4 + навык алхимии / 30 
Первое слагаемое округляется вверх, второе - вниз.

Второй бросок определяет качество зелий. Этот бросок делается для каждого зелья отдельно. Игрок озвучивает, какие зелья он желает изготовить.
Мастер определяет, какие зелья могут быть изготовлены. Порог, который должен быть пройден, определяется мастером.
Результат броска определяется по следующей формуле:
Бросок кубика + навык алхимии / 20 
Оба слагаемых округляются вверх.

На результат бросков могут влиять перки.
