Данный файл содержит описание процедуры создания персонажа
    При создании персонажа указывается его пол.

Раса
    При созднаии персонажа указывается его раса. Раса персонажа влияет на его внешность, бонусы к базовым атрибутам, а также даёт персонажу расовые способности.

Установка базовых атрибутов
    Значения всех базовых атрибутов равно 4. Игроку доступно 6 * 2 + 12 = 24 очка базовых атрибутов, которые он может потратить для их увеличения. 
1 вложенное очко повышает знчение атрибута на 1. При создании персонажа значение базового атрибута не может превысить 12.

Навыки и перки
    Все деревья навыков доступны персонажам любой расы. Величина навыка в каждом дереве и имеющиеся начальные перки зависят от расы персонажа. 
Все персонажи получают очки перков. Каждый перк для получения требует 1 очко перков. Уровень перка определяется его положением в дереве.
Персонаж может выбрать 4 основных навыка, которые он сможет задать при создании персонажа. Значение навыка может находиться в пределах от 0 до 100. 
При создании перснажа игрок получает 60 очков навыков, которые он может распределить между основными навыками. Значение навыка при создании персонажа 
не может быть задано более 30. Чем выше значение навыков, тем выше эффективность действий, связанных с этим навыком. 1 вложенное очко увеличивает 
значение навыка на 1.
    Каждая раса обладает 3 расовыми перками, уникальными для каждой расы. Эти перки начинают действовать сразу, не требуя очков для открытия.
    При создании персонажа игроку доступно 6 очков перков. При создании персонажа игрок не может развивать перки, уровень которых выше 1.

Предыстория
    Предысторию персонажа придумывает игрок. Это может дать герою дополнительные бонусы(перманентных эффекты, стартовые предметы, спутники, уникальные способности).
Получаемые бонусы определяются мастером.
