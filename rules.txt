1)Персонаж - существо, управляемое игроком или мастером.
     
2)Базовые атрибуты - параметры персонажа, представленные натуральными числами.
    Каждый атрибут своим значением определяет значения характеристик персонажа. Чем больше значение - тем лучше 
связанные с ним характеристики. Задаются в начале игры при создании персонажа. Игрок имеет в распоряжении 
определенное количество очков, которое он использует для повышения значения желаемых атрибутов. 
Одно вложенное очко увеличивает значение атрибута на единицу. В конце каждой игровой сессии игроку выдаётся 
определённое количество очков, которые игрок может потратить на повышение выбранных атрибутов. 
Значения атрибутов не могут быть больше определенной величины, за исключением тех случаев, когда первые изменяются 
по причине временных эффектов. Отрицательные значения атрибутов в последнем случае оказывают на персонажа негативный эффект, 
уникальный для каждого атрибута, который является крайне тяжелым для игрового процесса и который не исчезнет по 
истечению времени действия эффекта, вызвавшего понижение атрибута.
     
3)Характеристики. 
    Рассчитываются на основе базовых атрибутов. Выше значение — лучше. Отрицательные значения возможны для 
некоторых характеристик. Эффекты отрицательных значений включены в описание характеристики. Используются для расчета 
результатов действий персонажа.
     
4)Действия. 
Предпринимаются персонажем. Оканчиваются результатом. Чтобы совершить действие, 
игрок должен озвучить его мастеру. Если действие физически не может быть выполнено, оно не совершается.
После того, как действие озвучено, происходит бросок кубика. На результат также могут влиять следующие параметры:
    a.характеристики
    b.базовые атрибуты
    c.навыки

Каждый из перечисленных параметров влияет на результат броска, если этот параметр учитывается. Он может увеличивать 
результат броска. Полученный результат определяет последствия совершения действия, которые описываются игроку мастером. 
Чем выше результат, тем выше эффективность совершённого действия.  Для выполнения некоторых действий могут потребоваться 
определенные условия: порог для определённого базового атрибута, определенная степень владения необходимым навыком. 
Очередность выполнения действий определяется исходя из условий их выполнения.
     
5)Способности.
Специфическое действие или их совокупность, доступное игроку. Для применения способности могут требоваться определенные условия. Способности приобретаются персонажем в ходе продвижения по дереву навыков. При соблюдении всех условий персонаж может применить способность, в противном случае — нет.
     
6)Дерево навыков. 
Выбранное игроком направление развития, открывающее доступ к определенным способностям. Для получения способностей могут требоваться 
определенные условия. Выбор перка для развития делается игроком в конце сессии. В конце сессии игроку выдаётся 1 очко перков, 
за которые он может получать перки в дереве перков, а также 15 очков развития, которые игрок может использовать для развития навыка.
     
7)Эффекты.
Статусы, влияющие на материальные объекты. Могут быть получены и сняты в результате действий игрока, либо с течением времени. 
Для получения и снятия некоторых эффектов требуются определённые условия.
     
8)Предмет — материальная вещь. 
Персонаж может влиять на предметы, совершая действия. В результате действий может измениться перечень эффектов, наложенных на предмет. 
Предмет может накладывать эффекты без воздействия со стороны персонажа.

    e.Поиск сокровищ

    h.Вербовка сопартийцев

    l.Выполнение квестов
    m.Порядок выполнения действий
