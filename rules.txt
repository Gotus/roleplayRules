1)Расы. Существуют следующие расы: эльфы, великаны, дворфы, порождённые, люди, мыслители, тёмные эльфы;

    a.Эльфы (лесные).Давние обитатели лесов. Прирожденные охотники. Передают сие искусство из поколения в поколение. Клановый строй. Представители разных кланов могут находиться в разных взаимоотношениях, что может влиять на действия, совершаемые каждой из сторон. Высокие и немного худощавые, ловкие и неуловимые. Имеют массу тела ниже, чем масса человека того же роста. Эльфийские воины обладают острым слухом и зорким глазом. Мастера засад. Предпочитают дальний бой ближнему, потому в качестве оружия используют луки. Разбираются в лесных травах и зверях. Способны приручать последних. Часто сталкиваются с тёмными эльфами, с которыми торгуются, и от которых получают вещи, недоступные для приобретения другими путями.
    b.Тёмные эльфы (дроу). Жители пещер. Способны управлять землёй, за счёт чего распространились по всей земной коре, создав систему коридоров.Теократический строй. Поклоняются божеству, которое, как сами дроу считают, дало им эти силы. Торгуют с дворфами, эльфами и людьми. Скудный пещерный рацион сделал дроу всеядными, а беспросветная тьма - почти слепыми, оставив способность реагировать на свет, зато развила слух и эхолокацию. Живут в пещерах, но могут выбираться на поверхность земли в лесах. Длина тела примерно совпадает с человеческой. Худощавые. Умеют маскироваться, дабы не быть съеденными подземными тварями, некоторых из них дроу удаётся приручать и использовать. 
    c.Великаны. Кочевники. Имеют огромный рост и силу. Неповоротливы и тугоухи. Племенной строй. Каждое племя существует независимо от других и отчитывается лишь перед сёгуном. Территория разделена между племенами.
    d.Дворфы
    e.Люди (по идее не должны иметь ни преимуществ, ни недостатков (эталон))
    f.Мыслители (умные, но слабые)
    g.Порождённые
     
2)Персонаж - существо, управляемое игроком или мастером.
     
3)Атрибуты - параметры персонажа, представленные натуральными числами.
    Каждый атрибут своим значением определяет значения характеристик персонажа. Чем больше значение атрибута - тем лучше связанные с ним характеристики. Атрибуты задаются в начале игры при создании персонажа. Игрок имеет в распоряжении определенное количество очков атрибутов, которое он использует для повышения значения желаемых атрибутов. Одно вложенное очко атрибутов увеличивает значение атрибута на единицу. В конце каждой игровой сессии игроку выдаётся определённое количество очков атрибутов, которые игрок может потратить на повышение выбранных атрибутов. Значения атрибутов не могут быть больше определенной величины, за исключением тех случаев, когда атрибуты изменяются по причине временных эффектов. Отрицательные значения атрибутов в последнем случае оказывают на персонажа негативный эффект, уникальный для каждого атрибута, который является крайне тяжелым для игрового процесса и который не исчезнет про истечению времени действия эффекта, вызвавшего понижение атрибута.
     
4)Характеристики. 
    Рассчитываются на основе атрибутов и эффектов. Выше значение — лучше. Отрицательные значения возможны для некоторых характеристик. Эффекты отрицательных значений включены в описание характеристики. Используются для расчета результатов действий персонажа.
     
5)Действия. 
Предпринимаются персонажем. Оканчиваются результатом. Игрок озвучивает действие персонажа, Чтобы совершить действие, игрок должен озвучить его мастеру. Если действие физически не может быть выполнено, оно не совершается. Иначе, оно озвучивается игрокам. После того, как действие озвучено, происходит бросок кубика. На результат также могут влиять следующие параметры:
    a.характеристики
    b.атрибуты
    c.навыки

Каждый из перечисленных параметров влияет на результат броска, если этот параметр учитывается. Он может увеличивать результат броска. Полученный результат определяет последствия совершения действия, которые описываются игроку мастером. Чем выше результат, тем выше эффективность совершённого действия. 
Для выполнения некоторых действий могут потребоваться определенные условия: порог для определённого атрибута, определенная степень владения необходимым навыком. Очередность выполнения действий определяется исходя из условий их выполнения.
     
6)Способности. 
Специфическое действие или их совокупность, доступное игроку. Для применения способности могут требоваться определенные условия. Способности приобретаются персонажем в ходе продвижения по дереву навыков. При соблюдении всех условий персонаж может применить способность, в противном случае — нет.
     
7)Дерево навыков. 
Выбранное игроком направление развития, открывающее доступ к определенным способностям. Для получения способностей могут требоваться определенные условия. Выбор способности для развития делается игроком в конце сессии. В конце сессии игроку выдаётся определённое количество очков, за которые он может продвигаться по дереву навыков.
     
8)Эффекты. 
Статусы, влияющие на материальные объекты. Могут быть получены и сняты в результате действий игрока, либо с течением времени. Для получения и снятия некоторых эффектов требуются определённые условия.
     
9)Предмет — материальная вещь. 
Персонаж может влиять на предметы, совершая действия. В результате действий может измениться перечень эффектов, наложенных на предмет. Предмет может накладывать эффекты без воздействия со стороны персонажа.
